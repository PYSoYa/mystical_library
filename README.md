## 1. 프로젝트 명
### 신비한서재

## 2. 기획의도
#### 많은 작가들, 신진 작가를 포함하여 현업 작가들과 인기 작가들이 주로
#### 글 구독 서비스를 진행하고있습니다. 
#### 하지만 현재의 방식의 많은 문제점을 발견하였습니다. 
#### 작가만의 공간이 아닌 일반인들에게도 작가에 대한 꿈을 펼칠수 있는 
#### 공간을 만들어 주기 위해 이 서비스를 기획하게 되었습니다.
